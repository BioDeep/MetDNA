MetDNA requires the import of the following files for metabolite identifications, including:

  • A MS1 peak table (.csv format)
  • MS2 data files (.mgf or .msp format)
  • A table for sample information (.csv format)

Important Notes:
  1. "MS1 peak table" is a .csv file (separated by commas). The first column is the peak name ("name"). The second column is the mass-to-charge ratio ("mz"). The third column is the retention time ("rt"). The unit of retention time must be second (not minute). Other columns are peak abundances of MS1 peaks in each sample.

    IMPORTANT: the order and names of the first three columns (i.e., "name", "mz", and "rt") must be the same as the "Example of MS1 Peak Table".

  2. "Sample information" is a .csv file (separated by commas) that contains the group information of all the samples. The first column is the sample name ("sample.name"). The second column is the group name ("group").

    IMPORTANT: the order and names of the first two columns (i.e., "sample.name" and "group") must be the same as "Example of Sample information".

  3. "MS/MS data" are files in .mgf or .msp formats. Data conversion instruction can be found in help page.
